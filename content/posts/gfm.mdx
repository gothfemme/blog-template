## Callout

Also called admonitions, hints, etc. Supported via the Markdown directive syntax.

:::note
This is a note. It's of little consequence.
:::

:::tip
This is a tip. What an excellent idea.
:::

:::info
This is a piece of useful information.
:::

:::warning
This is a warning. How foreboding.
:::

:::danger
This is a dangerous thing. Don't do it.
:::
